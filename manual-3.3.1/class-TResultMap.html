Class TResultMap

TResultMap corresponds to mapping tag.

A TResultMap lets you control how data is extracted from the result of a query, and how the columns are mapped to object properties. A TResultMap can describe the column type, a null value replacement, and complex property mappings including Collections.

The can contain any number of property mappings that map object properties to the columns of a result element. The property mappings are applied, and the columns are read, in the order that they are defined. Maintaining the element order ensures consistent results between different drivers and providers.

The Class setClass() property must be a PHP class object or array instance.

The optional Extends setExtends() attribute can be set to the ID of another upon which to base this . All properties of the "parent" will be included as part of this , and values from the "parent" are set before any values specified by this .
