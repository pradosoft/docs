Class TTextProcessor

TTextProcessor class.

TTextProcessor is the base class for classes that process or transform text content into different forms. The text content to be processed is specified by setText Text property. If it is not set, the body content enclosed within the processor control will be processed and rendered. The body content includes static text strings and the rendering result of child controls.

Note, all child classes must implement TTextProcessor::processText() method.
