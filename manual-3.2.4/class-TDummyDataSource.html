Class TDummyDataSource

TDummyDataSource class

TDummyDataSource implements a dummy data collection with a specified number of dummy data items. The number of virtual items can be set via setCount Count property. You can traverse it using foreach PHP statement like the following,

foreach($dummyDataSource as $dataItem)
