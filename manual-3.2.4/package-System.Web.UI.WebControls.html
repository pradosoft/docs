TRepeatInfo TRepeatInfo class. TRepeatInfo represents repeat information for controls like TCheckBoxList. The layout of the repeated items is specified via setRepeatLayout RepeatLayout, which can be either Table (default), Flow or Raw. A table layout uses HTML table cells to organize the items while a flow layout uses line breaks to organize the items. The number of columns used to display the items is specified via setRepeatColumns RepeatColumns property, while the setRepeatDirection RepeatDirection governs the order of the items being rendered.
