Class TWizardNavigationContainer

TWizardNavigationContainer class.

TWizardNavigationContainer represents a control containing a wizard navigation. The navigation may contain a few buttons, including getPreviousButton PreviousButton, getNextButton NextButton, getCancelButton CancelButton, getCompleteButton CompleteButton.
