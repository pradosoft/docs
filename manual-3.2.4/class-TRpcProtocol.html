Class TRpcProtocol

TRpcProtocol class

TRpcProtocol is the base class used to implement a protocol in a TRpcService. Prado already implements two protocols: TXmlRpcProtocol for Xml-Rpc request and TJsonRpcProtocol for JSON-Rpc requests.
