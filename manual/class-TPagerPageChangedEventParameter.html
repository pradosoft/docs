Class TPagerPageChangedEventParameter

TPagerPageChangedEventParameter class

TPagerPageChangedEventParameter encapsulates the parameter data for PageIndexChanged event of TPager controls.

The getCommandSource CommandSource property refers to the control that originally raises the OnCommand event, while getNewPageIndex NewPageIndex returns the new page index carried with the page command.
