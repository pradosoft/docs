Class TActiveRecordChangeEventParameter

TActiveRecordChangeEventParameter class

TActiveRecordChangeEventParameter encapsulates the parameter data for ActiveRecord change commit events that are broadcasted. The following change events may be raise: TActiveRecord::OnInsert, TActiveRecord::OnUpdate and TActiveRecord::OnDelete. The setIsValid IsValid parameter can be set to false to prevent the requested change event to be performed.
