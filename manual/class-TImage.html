Class TImage

TImage class

TImage displays an image on a page. The image is specified via the setImageUrl ImageUrl property which takes a relative or absolute URL to the image file. The alignment of the image displayed is set by the setImageAlign ImageAlign property. To set alternative texts or long description of the image, use setAlternateText AlternateText or setDescriptionUrl DescriptionUrl property, respectively.
